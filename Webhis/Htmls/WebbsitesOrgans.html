Behind The Website

Behind every website there is long textfiles containing the commands needed to make the website look like it does. Different elements and functions create animations and styles.

The base code for every website is HTML (HyperText Markup Language) which is basically just a long textfile. CSS (Cascading Style Sheet) is what gives the websites the color and the ability of customization. JS (Javascript) is used for scrips on websites and can also be used with animations and buttons.

HTML

HyperText Markup Language

HTML stands for hypertext markup language and is the code that is the base of every single website in the world. Without HTML a website would not exist. Although by only using HTML and no CSS or JS then the website would look very boring. Click here to see an example.

What's the use for it?

HTML is used for the writing, links, pictures, sound, and video on the webpage and tells the web browser what the website should look like. It also contains the metadata information which is for example character encoding, content, author, and keywords.

HTML 5

HTML was made by  World Wide Web Consortium (W3C). The current version of the language is HTML 5 (2022). It works by using elements to create the websites. An example of very simple HTML code can be seen in the picture.

Cascading Style Sheet

CSS

CSS (Cascading Style Sheet) was created by Håkon Wium Lie on 10 October 1994 and is the language used by websites to give their distinct and well-made design. Without CSS, websites look like plain text files and are very boring. CSS can be used to change Fonts, Colors, Layouts, and Shapes. Therefore, This allows for the separation of content and presentation and can make the experience on a website much more enjoyable.

Furthermore, it allows for an adaptable presentation of the website so it works on all sorts of devices either it being large or small screens. Another way the language is adaptable is by it being able to be used with any XML-based markup language such as SVG, MathML or XHTML.

Perks

With CSS, the amount of code in the document itself is significantly reduced, and makes the pages load faster in the browser by creating CSS stylesheet files. CSS can either be prioritized differently depending on if something is said in the CSS file or if it's said in the HTML file.

CSS 1, 2, 3

CSS 1 contains information about relatively simple formatting such as background colors, background images, fonts, and text alignment. CSS 2 goes a step further and provides the opportunity to design entire documents with positioned rectangular areas (DIVs, boxes) that were previously only possible with tables. CSS 3 includes several effects that today are not fully supported in all browsers but can still be used.

Javascript

JS

Javascript is a language that is commonly used on websites when producing dynamic effects on the website. JS can be used for animations, video playback, interactive menus, and much more. You can also change different elements on the website by for example changing the background color when you click a picture.

Brendan Eich

Javascript was created by Brendan Eich who was an American developer. He wanted to create a language that would be used as a "glue language" for informal programmers and designers. Javascript allowed programmers to use JS to put together components and automate interactions.
